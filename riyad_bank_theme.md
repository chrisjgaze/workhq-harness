# Riyad Bank – Inspired Color Theme Guide

## 🎯 Purpose
This file defines a practical **Riyad Bank-inspired** color system for:
- Presentations (PPTX)
- Infographics
- HTML dashboards
- Generated visuals
- Customer/account briefing slides

> Note: This is an **inspired working theme**, not an official Riyad Bank brand guideline. It is based on the visible Riyad Bank website and logo styling, with colours selected to create a professional, bank-appropriate look and feel.

---

# 🎨 1. PRIMARY COLORS (Core Brand – Use First)

The Riyad Bank identity is strongly associated with **green**, supported by **blue** and clean white space. Prioritise these colours for slides, dashboards, headers and major visual elements.

| Name | Hex | Usage |
|------|-----|------|
| Riyad Green | `#00843D` | Primary brand accent, key icons, active states |
| Deep Riyad Green | `#006B31` | Headers, strong emphasis, dark buttons |
| Dark Forest Green | `#004F2A` | Executive titles, footer bands, serious sections |
| Fresh Green | `#00A651` | Highlights, positive outcomes, success indicators |
| Soft Green | `#DFF3E8` | Light panels, background fills, subtle sections |
| Pale Green | `#EEF8F2` | Page backgrounds, low-emphasis cards |
| Riyad Blue | `#005EB8` | Digital banking, links, secondary accents |
| Deep Blue | `#003B73` | Corporate depth, charts, dark contrast |

✅ Use for:
- Main slide titles
- Navigation bars
- Icons
- Primary callouts
- Positive metrics
- Dashboard active states

# 🧠 DESIGN RULES

## 1. Priority Order
1. Riyad Green family
2. Blue / Teal support colours
3. Neutrals
4. Gold / Purple / Red accents only when needed

---

## 2. Do NOT
❌ Overuse bright green on large backgrounds  
❌ Mix too many accents on one slide  
❌ Use red unless there is a genuine risk or warning  
❌ Use gold as a general decoration  
❌ Make dashboards too dark unless the slide is intentionally executive-style

# 🛠️ USAGE FOR AUTOMATION

When generating:
- **PPTX** → Use dark green headers, white content cards, green/blue accents.
- **HTML** → Use CSS variables below for consistent styling.
- **Charts** → Start with green, then blue, then teal.
- **Icons** → Use green for current/primary, blue for digital/supporting, gold for value, red for risks.
- **Infographics** → Use clean whitespace with green titles and restrained accent colours.
