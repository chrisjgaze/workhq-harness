# SS&C Blue Prism – Color Theme Guide

## 🎯 Purpose
This file defines the official color system for SS&C Blue Prism styled outputs, including:
- Presentations (PPTX)
- Infographics
- HTML dashboards
- Generated visuals

The goal is **consistency, clarity, and brand alignment**.

---

# 🧠 DESIGN RULES

## 1. Priority Order
1. Primary Blues
2. Secondary Blues/Teals
3. Neutrals
4. Accent Colors (only when needed)

---

## 2. Do NOT
❌ Overuse yellow/gold  
❌ Mix too many color families on one slide  
❌ Use purple unless highlighting something special  

---

## 3. Recommended Combinations

### Clean Corporate Slide
- Background: #FFFFFF or #E6F1FA
- Title: #004778
- Content: #1F2734
- Accent: #0077C8

### Data / Dashboard
- Primary: #005FA0
- Secondary: #40ADBD
- Support: #6D94C2

### AI / Innovation Slide
- Base: Blue palette
- Highlight: Purple (#60499E)

### ROI / Value Slide
- Base: Blue palette
- Highlight: Gold (#F5AB0A)

---

# 🛠️ USAGE FOR AUTOMATION

When generating:
- PPTX → Use primary blues for headers
- HTML → Use CSS variables from primary palette
- Charts → Start with primary → expand into secondary
- Icons → Match left (dark blue) / right (light blue or accent)

---

# 📌 SUMMARY

- **Blues = Core identity**
- **Teals = Depth & variation**
- **Purple = Intelligence / AI**
- **Gold = Value / attention**
- **Greys = Structure**

Keep it clean. Keep it consistent. Keep it SS&C.
